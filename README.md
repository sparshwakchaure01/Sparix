# Sparix
SPARIX is a modern Student Operating System built to help students study smarter, stay organized, and accelerate academic and career growth. It combines productivity tools, college utilities, career resources, and learning workflows into one clean, mobile-first platform powered by HTML, CSS, and JavaScript.
